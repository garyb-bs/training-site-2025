Accessibility Issues Summary

This page intentionally contains key accessibility issues for training purposes. These issues can be easily detected by automated accessibility testing tools.

Key Accessibility Issues for Automated Testing:

  • No skip links for keyboard users
  • Poor heading structure (no H1, missing headings)
  • Images without alt text
  • Poor color contrast ratios
  • Color-only information (no text alternatives)
  • Poor keyboard navigation and focus management
  • No form labels or proper associations
  • Poor tab order (tabindex misuse)
  • Clickable elements without keyboard support
  • No form validation feedback
